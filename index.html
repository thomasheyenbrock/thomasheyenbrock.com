About this website

You might have noticed that this website looks quite plain. That's because it is, intentionally.

I do know how to use the most performant frameworks and the most slick style libraries and the funkiest fonts. But would it really make this website so much better?

The simple purpose of this website to show text and link to external platforms. Just HTML with a tiny bit of CSS does that perfectly.

I'd rather use the time for something more impactful. Simplicity for the win.
